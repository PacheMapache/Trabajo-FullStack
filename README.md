# Evento

- OnClick -> Click del usuario
- OnChange -> Cuando escribe un input
- OnSubmit -> Cuando envía un formulario
- OnKeyDown -> Cuando pulsa una tecla
- OnMouseEnter -> Pasa el mouse

- useState -> hook
    - Crea una variable de estado
    - Crea una función para actualizar
    - Le dice a React cuando cambiar

const [valor, setValor]=useState(valorInicial);

const [contador, setContador]= useState(0);

- Contador -> Valor actual
- setContador -> función que actualiza
- 0 -> Valor Inicial
